Implementation Notes

Copy-Paste Ready

All CSS custom properties are defined in :root. Click any hex value on this page to copy it. Drop the variable block into your stylesheet and go.

No Sub-Styles

Beer categories have sub-style mapping (IPA → Pale Ale, etc). The four new beverage categories do not. Rosé, Merlot, Champagne — they're all --style-wine. Keep it flat.

Same Rules Apply

Content-vs-chrome separation. Max 2–3 style colors per viewport. Functional colors in their own lane. Nothing changes — we just have more colors to work with.

Jordan — Handoff

The new CSS variables follow the exact same naming convention. --style-[category], -light, -dark-soft. No surprises. Add them below the existing beer style block in the theme file.
