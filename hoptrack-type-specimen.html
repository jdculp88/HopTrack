06 — Implementation
For Jordan & Alex
Copy-paste ready. These are the only fonts we use. If it's not on this page, it doesn't go in the product.
/* ── Fontshare CDN ── */
@import url('https://api.fontshare.com/v2/css?
  f[]=satoshi@300,400,500,700,900
  &f[]=general-sans@300,400,500,600,700
  &display=swap');

/* ── Google Fonts CDN ── */
@import url('https://fonts.googleapis.com/css2?
  family=JetBrains+Mono:wght@300;400;500;700
  &display=swap');

/* ── CSS Variables ── */
--font-primary: 'Satoshi', sans-serif;
--font-display: 'General Sans', sans-serif;
--font-data:    'JetBrains Mono', monospace;
Licensing
All three fonts are free for commercial use. No licensing fees. No attribution required in-app.
Performance
Use display=swap on all imports. Subset if native — full charset for web.
Fallbacks
System sans-serif stack. Don't load Helvetica as fallback — use the OS default to avoid FOUT.
