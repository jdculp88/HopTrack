03 — Decision Tree
Which Background For What
Don't think about it — follow the map. Card type determines background. No freestyle.
01
Is it celebrating something?
Achievement unlocked, badge earned, milestone hit → Radiating Arcs from the icon. Tint to match the tier (bronze/silver/gold). The arcs say "something special happened here."
02
Is it happening right now?
Live session, active check-in, someone drinking right now → Glow + Pulse. Green dot pulses. Background has a faint green wash. The card literally feels alive.
03
Is it showing numbers?
Your Numbers, session stats, profile metrics → Grid Lines. Faint intersecting lines create a subliminal chart/graph feel. Data has structure; the background should too.
04
Is it editorially featured?
Beer of the Week, staff pick, curated spotlight → Warm Spotlight. Dual radial glow from upper-left. Gold border. This content was chosen — it should feel chosen.
05
Is it seasonal or limited?
Seasonal release, limited drop, time-bound beer → Organic Blob. Soft irregular shape in the corner, tinted to the beer's style color. Feels ephemeral, natural, not-forever.
06
Is it a collection or category?
Curated collection, style group, "The Dark Side" → Left-Edge Bar + Tint. 4px color bar on the left, subtle gradient bleed. Instant style identification on scroll.
07
Is it asking you to do something?
Plan a HopRoute, start a session, explore nearby → Topo Lines (for HopRoute) or Attention Gradient (for alerts). CTAs get directional energy. Notifications get warmth.
08
Is it about a person?
Friend joined, someone followed you, social action → Nothing. Clean white. The avatar IS the visual. Background would compete with the human element. Restraint is the design here.
04 — Implementation
Pure CSS — No Images
Every background is built from gradients, borders, and pseudo-elements. Zero image loads. Zero performance cost. Jordan can copy these directly.
/* Achievement — Radiating arcs from icon */ .card-achievement { background: radial-gradient(circle at 28px 50%, rgba(160,120,80,.06) 0%, rgba(160,120,80,.02) 40%, transparent 65%); } .card-achievement::before { /* Inner arc */ width: 80px; height: 80px; border-radius: 50%; border: 1px solid rgba(160,120,80,.08); } .card-achievement::after { /* Outer arc */ width: 110px; height: 110px; border: 1px solid rgba(160,120,80,.04); } /* Live Session — Glow + pulse */ .card-live { background: radial-gradient(ellipse at 50% 40%, rgba(72,160,88,.06) 0%, transparent 60%); border-color: rgba(72,160,88,.1); } /* Stats — Subtle grid */ .card-stats { background: repeating-linear-gradient(90deg, transparent, transparent 23%, rgba(183,82,47,.025) 23%, rgba(183,82,47,.025) 23.5%, transparent 23.5%), repeating-linear-gradient(0deg, transparent, transparent 48%, rgba(183,82,47,.025) 48%, rgba(183,82,47,.025) 49%, transparent 49%); } /* Featured — Warm spotlight */ .card-featured { background: radial-gradient(ellipse at 30% 30%, rgba(200,148,58,.08) 0%, transparent 50%), radial-gradient(ellipse at 80% 80%, rgba(183,82,47,.04) 0%, transparent 40%); } /* HopRoute CTA — Topo lines */ .card-hoproute { background: repeating-linear-gradient(165deg, transparent, transparent 18px, rgba(183,82,47,.025) 18px, rgba(183,82,47,.025) 19px); } /* Collection — Left-edge bar + gradient bleed */ .card-collection::before { width: 4px; background: var(--style-color); } .card-collection::after { background: linear-gradient(90deg, rgba(var(--style-rgb), .04) 0%, transparent 40%); } /* Seasonal — Organic blob */ .card-seasonal::before { border-radius: 40% 60% 55% 45% / 50% 40% 60% 50%; background: rgba(var(--style-rgb), .06); } /* Social — intentionally nothing */ .card-social { background: var(--warm-white); }
