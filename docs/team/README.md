## How we work

From [root CLAUDE.md](../../CLAUDE.md):

- **Decision making:** Joshua owns vision and go/no-go; Morgan owns program priorities; Sage owns sprint execution; Jordan owns technical strategy; Avery enforces code quality.
- **Push to main:** no PR gates. The founder trusts the team.
- **Retros every sprint:** fun first, honest always. Roasts preserved. Format lives in the [hoptrack-retro-format skill](../../.claude/skills/hoptrack-retro-format/SKILL.md).
- **Sprint close:** 6-step ceremony triggered by Joshua saying "close the sprint". Lives in the [sprint-close skill](../../.claude/skills/sprint-close/SKILL.md).
