## Step 3: Do All the Things

Over the weekend, try to do each of these at least once:

- [ ] Start a session at a brewery (or "at home")
- [ ] Log 2+ beers in a single session with ratings and notes
- [ ] End the session and go through the session recap
- [ ] Rate beers you haven't rated before (tap the stars)
- [ ] Leave a brewery review (star rating + comment)
- [ ] Check the Friends tab — see your friends' sessions
- [ ] Tap the Cheers button on someone's session card
- [ ] Leave a comment on someone's session
- [ ] Check the Discover tab — BOTW, Trending, Events, Seasonal
- [ ] Check the You tab — stats, Taste DNA, recent achievements
- [ ] Visit a brewery page from the Explore tab
- [ ] Add a beer to your Wishlist
- [ ] Check the notification bell for activity

## Step 4: Note What Works & What Doesn't

For Monday's debrief, write down:

1. **What felt great?** — moments where the app clicked
2. **What was confusing?** — where you got stuck or didn't know what to do
3. **What was broken?** — actual bugs (screenshot if possible)
4. **What's missing?** — features you expected but couldn't find
5. **What would you show a brewery owner?** — the "wow" moments
